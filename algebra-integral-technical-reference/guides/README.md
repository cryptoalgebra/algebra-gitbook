# Guides

This section provides hands-on guides for building with Algebra Integral. Two areas are covered:

**Contract Development** — step-by-step walkthroughs for integrating with Algebra Integral from your own smart contracts:

* [Environment Setup](contract-development-intro.md) — set up a Hardhat development environment with the Algebra packages
* [Swaps](swaps/README.md) — execute single-hop and multi-hop swaps
* [Providing Liquidity](providing-liquidity/README.md) — mint positions, collect fees, and manage liquidity
* [Flashloans](flashloans/README.md) — write contracts that use Algebra flash transactions

**Plugin Development** — guides for creating, testing, and deploying custom Algebra plugins:

* [Plugin Development](custom-plugins/plugin-development.md)
* [Plugin Testing](custom-plugins/plugin-testing.md)
* [Plugin Deployment](custom-plugins/plugin-deployment.md)
